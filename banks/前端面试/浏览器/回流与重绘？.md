回流和重绘本质上是浏览器渲染页面的步骤之一

浏览器绘制页面的步骤如下：

1. 构建 DOM 树
2. 构建 CSSOM 树
3. 构建渲染树
4. 布局：计算元素的位置、大小
5. 绘制：计算元素的颜色等
6. 合成

- 回流（Reflow）：就是重新跑一遍「布局」，改变几何（尺寸/位置）触发重新布局，开销大。
- 重绘（Repaint）：就是重新跑一遍「绘制」，改变外观但不改布局。
- transform / opacity 是仅有的两个 compositor-only 属性：当元素**已被提升为独立合成层**时，对其做动画可跳过 layout/paint；若未提升仍可能触发 paint。

优化：批量改、读写分离避免强制同步布局、用 class 批量改样式、用 DocumentFragment。
