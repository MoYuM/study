# 节流（throttle）

固定时间间隔内最多执行一次（如滚动、resize）。

```jsx
function throttle(fn, interval) {
  let last = 0;
  return function (...args) {
    const now = Date.now();
    if (now - last >= interval) {
      last = now;
      fn.apply(this, args);
    }
  };
}
```

也可用定时器版（`!timer` 才设定时器）；两版结合可兼顾"首次立即执行"与"最后一次也执行"。

# 防抖（debounce，含 immediate）

事件停止触发 n 毫秒后才执行，连续触发则重新计时（如搜索联想）。两个易漏点：**1）核心用 `setTimeout`；2）用 `apply` 转发 `this` 和参数**。

```jsx
function debounce(fn, delay = 1000, immediate = false) {
  let timer = null;
  return function (...args) {
    if (timer) clearTimeout(timer);
    // immediate=true 且当前没有定时器（即首次触发）→ 立即执行
    if (immediate && !timer) fn.apply(this, args);
    timer = setTimeout(() => {
      timer = null; // 关键：清空，否则 immediate 只会触发一次，之后再也不会"立即执行"
      if (!immediate) fn.apply(this, args);
    }, delay);
  };
}
```
