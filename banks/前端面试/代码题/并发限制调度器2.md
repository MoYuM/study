控制同时执行的异步任务数量（如最多并发 2 个）。

思路：维护运行计数和等待队列；addTask 返回 Promise，未达上限立即执行，达上限则入队；每个任务完成后计数减一并从队列取下一个执行。

```jsx
class Scheduler {
  constructor(max) { this.max = max; this.count = 0; this.queue = []; }
  add(task) {
    return new Promise((resolve) => {
      const run = () => {
        this.count++;
        task().then(resolve).finally(() => {
          this.count--;
          if (this.queue.length) this.queue.shift()();
        });
      };
      this.count < this.max ? run() : this.queue.push(run);
    });
  }
}
```
