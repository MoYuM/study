用 Map 建 `id -> node` 映射，遍历数组将每个节点挂到其 parent.children，pid 为根的收集为顶层。

```jsx
function listToTree(list) {
  const map = new Map(), roots = [];
  list.forEach(n => map.set(n.id, { ...n, children: [] }));
  list.forEach(n => {
    const node = map.get(n.id);
    if (n.pid && map.has(n.pid)) map.get(n.pid).children.push(node);
    else roots.push(node);
  });
  return roots;
}
```

时间复杂度 O(n)。
