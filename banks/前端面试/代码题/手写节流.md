节流：固定时间间隔内最多执行一次（如滚动、resize）。

```jsx
function throttle(fn, interval) {
  let last = 0;
  return function (...args) {
    const now = Date.now();
    if (now - last >= interval) {
      last = now;
      fn.apply(this, args);
    }
  };
}
```

也可用定时器版；两版结合可兼顾首次与最后一次。
