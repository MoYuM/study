- Flex：父 `display:flex; align-items:center; justify-content:center;`
- Grid：父 `display:grid; place-items:center;`
- 绝对定位 + transform：`position:absolute; top:50%; left:50%; transform:translate(-50%,-50%);`（不需知道自身尺寸）
- 绝对定位 + margin auto：`position:absolute; inset:0; margin:auto;`（需定宽高）
- 单行文本：`line-height` 等于容器高度。
