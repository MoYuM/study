`flex: 1` = `flex-grow:1 flex-shrink:1 flex-basis:0%`，表示等分剩余空间。

- 主轴对齐：justify-content；交叉轴：align-items。
- 三个子属性：grow（放大比例）/ shrink（缩小比例）/ basis（基准尺寸）。
