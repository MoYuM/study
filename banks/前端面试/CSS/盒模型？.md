主要是两种计算元素宽高的方式

四层结构：content -> padding -> border -> margin

- content-box（标准）：width 不含 padding/border。
- border-box：width 包含 padding/border，布局更可控。

用 `box-sizing` 切换。盒子由内到外：content → padding → border → margin。
