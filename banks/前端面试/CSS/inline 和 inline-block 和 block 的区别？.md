---
题目: "inline 和 inline-block 和 block 的区别？"
分类: CSS
频率: 低频
id: c5062df9-be09-4f46-8d77-6233168e6c08
---
- inline 宽度不够时，自动切断换行，inline-block 不会切断
- inline 不能设置宽高 ⇒ inline-block 和 block 可以
- block 独占一行
- inline 元素不可以设置上下的 margin 和 padding
