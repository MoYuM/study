题目: "响应式布局方案？"
分类: CSS
频率: 高频
id: 2965ed57-6149-419c-b1c5-d5916cef4d45
---
- 媒体查询 `@media` 按断点切换布局。
- 相对单位：rem / em / vw / vh / %。
- 弹性布局：flex / grid。
- rem 方案：JS 动态设置 html 根字号（或 vw），配合 postcss-px-to-viewport 自动换算。
- viewport meta、图片 `srcset`/`picture` 响应式图片。

移动端常用 rem 或 vw 方案；多端适配优先 flex/grid + 媒体查询。
