## 三种主流方式

```js
// 1. Set（最简洁，日常首选）O(n)
const unique = arr => [...new Set(arr)];

// 2. Map 记录（需自定义 key 逻辑时用）O(n)
function unique(arr) {
  const seen = new Map();
  return arr.filter(item => !seen.has(item) && seen.set(item, true));
}

// 3. filter + indexOf（语义清晰，但 O(n²)，大数据不推荐）
const unique = arr => arr.filter((item, i) => arr.indexOf(item) === i);
```

## 对比

| 方式 | 时间复杂度 | 适用场景 |
|------|-----------|---------|
| `Set` | O(n) | 基本类型，简单去重首选 |
| `Map` | O(n) | 需自定义去重 key（如对象按字段去重） |
| `indexOf` | O(n²) | 数据量小、可读性优先 |

## 对象数组按字段去重

```js
const uniqueById = arr => {
  const map = new Map();
  arr.forEach(item => map.set(item.id, item));
  return [...map.values()];
};
```
