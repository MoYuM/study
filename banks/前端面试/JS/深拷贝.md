深拷贝递归复制对象所有层级，拷贝后与原对象完全独立。

手写要点：基本类型直接返回；用 WeakMap 记录已拷贝对象解决循环引用；区分数组与对象；Date / RegExp / Map / Set 等特殊类型单独处理；函数通常直接引用。

原生方案 `structuredClone()` 可处理循环引用与多数内置类型，但不支持函数和 DOM 节点；`JSON.parse(JSON.stringify())` 简单但会丢失函数、undefined、Symbol，且无法处理循环引用。
