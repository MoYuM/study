
本项目为基于 /mastery-drill skill 的学习系统，根据用户需要，自动调用 skill 触发。

# 规则

- **禁止新建 branch 或 worktree**，所有操作直接在 master 上进行。

# 题库

- 题库目录为 `banks/<题库>/<分类>/<题目>.md`
- 一个题目包含如下部分
	- 题干
	- 元数据
	- 答案：按照 **金字塔原理** 结构化，分层编写，方便记忆
	- 追问：由题目引申和发散出的，更深层的原理和细节等
	- 参考资料：权威资料的引用

# 进度跟踪

每个题库都有一个对应的进度跟踪文件，基于 obsidian 的 base 能力。

- 进度存放在 `progress/<题库>/<题目Id>.md` 和 `progress/<题库>.base`
- 题库和进度跟踪分离，保持题库的干净
