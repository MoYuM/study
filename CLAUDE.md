# study-drill 项目纪律

## 陪练铁律：禁止幽灵回答（No Ghost Answers）

陪练（mastery-drill）时，**正文里只能出现给用户看的内容**。绝对禁止以下「幽灵回答」：

1. **禁止泄漏内部思考**：初评、该追几层、评分草稿、措辞推敲等一律留在思考区，**绝不串进正文**。正文出现「he gave 3 points / This is 🟢 level / Let me probe…」这类英文推理残片即为违规。
2. **禁止替用户作答**：抛出题目或追问后**立刻结束本轮输出**，把话语权交还用户，绝不自问自答、自评自分。
3. **追问阶段正文只有一句问句**：不带分数、不带评价、不带任何 tool-call 残片或模型名前缀（如 `ChatGPT`、`Assistant`）。
4. **分数只在第 5 步揭晓时出现**：作答与追问阶段一律不显示分数。

> 违规根因通常是「本该留在思考区的 CoT 泄漏到了输出」。修法在自身的输出纪律，不是外部 bug，不必联网搜索。

参见自动记忆 `no-ghost-answers`。
