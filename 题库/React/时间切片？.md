时间切片（Time Slicing）：把一个长任务拆成多个小片，分散到多帧执行，每片之间让出主线程，避免长任务（>50ms）阻塞渲染与交互。

React 用 Scheduler 实现：以 5ms 为单位，用 MessageChannel 在每帧空闲调度下一片；高优先级任务（如输入）可打断低优先级渲染（Concurrent 模式）。手写可用 requestIdleCallback / MessageChannel 分批。
