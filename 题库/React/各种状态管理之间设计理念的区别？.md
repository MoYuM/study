- Redux：单一 store、状态不可变、纯函数 reducer、中间件处理副作用；可预测、可调试，但模板代码多（RTK 简化）。
- MobX：响应式 + 可变状态，自动追踪依赖，心智负担小但可控性弱。
- Zustand：基于 hook、极简、无 Provider、按需订阅。
- Recoil / Jotai：原子化，细粒度响应式更新。
- Vuex / Pinia：Vue 生态，Pinia 更轻、TS 友好。

选型看：项目规模、是否需要严格可追溯、细粒度更新与心智模型偏好。
