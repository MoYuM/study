- 首选 CSS `position: sticky; top: 0;`：滚动到阈值后自动吸附，代码最简；需至少一个 inset（top/bottom 等）非 auto。
- sticky 相对「最近的滚动祖先」定位：若某祖先用 overflow: hidden/auto/scroll 裁剪内容或不提供可滚动空间，会表现为失效/被裁剪；祖先足够高且不裁剪时仍正常工作。
- 兼容/复杂场景：监听 scroll，到阈值切换 fixed + 占位补高；用 IntersectionObserver 更高效。
