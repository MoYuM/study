分三层优化：

- **加载**：资源压缩（gzip/brotli）、HTTP 缓存、CDN、代码分割与按需加载、图片懒加载与 WebP、预加载 preload/prefetch、Tree-shaking。
- **渲染**：减少回流重绘、虚拟列表、骨架屏、SSR/同构、关键 CSS 内联、首屏优先。
- **运行时**：防抖节流、Web Worker 处理重计算、避免长任务、合理使用 requestAnimationFrame。

衡量指标：Web Vitals（FCP / LCP / INP / CLS），用 Performance 面板与 Lighthouse 定位瓶颈。
