- Symbol（ES6 / ES2015）：唯一值，常用作对象的唯一属性键、定义常量，内置如 Symbol.iterator。
- BigInt（**ES2020**，并非 ES6）：表示超过 Number.MAX_SAFE_INTEGER 的大整数，字面量加 n（如 10n）。

补充：Set / Map / WeakMap / WeakSet 是 ES6 新增的内置结构（不是原始类型）。
