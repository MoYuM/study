- cookie：约 4KB，随同源请求自动携带，可设过期/HttpOnly/Secure/SameSite，多用于鉴权。
- localStorage：5–10MB，持久化，同源共享，需手动清除。
- sessionStorage：会话级，标签页隔离，关闭即清。
- IndexedDB：大容量、结构化、异步，适合离线数据。
- Cache API / Service Worker：缓存资源做离线。

按容量、生命周期、是否随请求发送来选型。
